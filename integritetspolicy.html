Integritetspolicy

Introduktion

Välkommen till Matsyn! Vi är engagerade i att erbjuda dig detaljerad och korrekt information om livsmedelsinnehåll. Din integritet är vår prioritet, och vi vill säkerställa att du förstår hur vi hanterar din information.

Information vi samlar in

Matsyn samlar inte in någon personlig information från sina användare. Vår applikation är utformad för att erbjuda information om livsmedelsinnehåll utan behov av att samla in eller lagra personlig data.

Användning av Livsmedelsverkets API

Matsyn använder Livsmedelsverkets API, tillhandahållet av den svenska Livsmedelsverket, för att hämta och leverera information om livsmedelsprodukter. Detta API är en värdefull resurs för att få data relaterad till livsmedelsinnehåll.

Så här använder vi Livsmedelsverkets API:

  • Data som tillhandahålls av API:t: Vi använder API:t för att hämta information om olika livsmedelsprodukter, inklusive näringsdata, ingredienser och andra relevanta detaljer. Denna data används för att ge dig exakt livsmedelsinformation inom appen.
  • Ingen personlig data åtkomst: Livsmedelsverkets API tillhandahåller inte tillgång till någon personlig information. Den data som erhålls genom API:t är begränsad till livsmedelsrelaterad information och inkluderar ingen användarspecifik data.

Användning av data

Vår app är utformad för att enbart ge information om livsmedelsinnehåll. Vi samlar inte in, använder eller lagrar någon personlig data. Matsyns primära funktion är att presentera exakt livsmedelsrelaterad innehåll för att öka din kunskap om de produkter du konsumerar.

Cookies och spårning

Matsyn använder inte cookies eller någon spårningsteknik för att samla in data om din användning eller preferenser. Det finns ingen spårning av dina aktiviteter utöver vad som är nödvändigt för att visa appens innehåll.

Tredjepartstjänster

Förutom användningen av Livsmedelsverkets API för livsmedelsinformation integrerar Matsyn inte med några tredjepartstjänster som kräver åtkomst till din personliga data. Vårt fokus förblir på att erbjuda innehåll och funktionalitet som är relevant för livsmedelsinformation.

Säkerhet

Vi är engagerade i att upprätthålla säkerheten för Matsyn och den information den tillhandahåller. Eftersom vi inte samlar in någon personlig data, finns det ingen personlig information som är i riskzonen. Vi genomför lämpliga åtgärder för att säkerställa säkerheten och integriteten hos den information som levereras genom appen.

Ändringar i denna policy

Vi kan uppdatera denna integritetspolicy från tid till annan för att återspegla förändringar i våra rutiner eller för att uppfylla lagliga krav. Eventuella uppdateringar kommer att göras tillgängliga inom appen, och vi uppmuntrar dig att regelbundet granska denna policy.

Kontakt

Om du har några frågor eller funderingar angående denna integritetspolicy eller Matsyns rutiner, vänligen kontakta oss på kontakt@matsyn.se.
